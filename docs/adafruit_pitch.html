PROJECT SHOWCASE // RP2040 & CIRCUITPYTHON

Motorized Coaxial TIG Welding Pen

Hi Adafruit Team,

I wanted to share a custom fabrication tool I've built that leverages the power of CircuitPython and the RP2040 in a high-stress, real-world application: Project Delta Pen (DP-1).

Figure 1.0: DP-1 Coaxial Wire Feeder powered by CircuitPython

How We Built It

Processor: RP2040 microcontroller running a low-latency state machine.
Control Loop: CircuitPython script mapping analog FSR grip force directly to step intervals.
Safety: GPIO hardware safety interrupt (Pulse-Dab Switch) to halt stepper drivers instantly.
